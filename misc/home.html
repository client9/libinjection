libinjection

Libinjection: advanced sqli detection. Try it now on the diagnositcs page.

About

  • Open source, code is at github
  • BSD License
  • Standard C code, compiles on linux/unix/bsd, mac and windows
  • Over 500k inspections per second on commodity hardware
  • No threads and no memory allocation - only 500 bytes on stack is used
  • Small, under 1000 lines of code in only three files ( 1, 2, 3)
  • No external library dependencies

Quality, Accuracy, Precision

Continuous integration using Jenkins is publically accessible at https://libinjection.client9.com/jenkins/view/libinjection/.

Tests include:

  • built and tested using gcc and clang (windows tested via modsecurity)
  • static analysis using cppcheck and clang's static analyzer
  • tested against 50,000 sqli samples
  • tested against previous and current false positives
  • over 100+ unit tests, run under valgrind for memory problems
  • code coverages from unit tests and sqli samples
